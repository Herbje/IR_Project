A 104-key PC US English keyboard layout with Prt Sc circled. Print screen (often abbreviated Print Scrn, Prnt Scrn, Prt Scn, Prt Scr, Prt Sc or Pr Sc) is a key present on most PC keyboards. It is typically situated in the same section as the break key and scroll lock key. The print screen may share the same key as system request.
